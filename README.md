# Revit Example Plugin

Тестовый плагин для Revit. Демонстрирует навыки работы с Revit API на C#.

## Что внутри
- **Создать РН** — создаёт рабочие наборы по выбранному .txt файлу.
- **Записать помещения** — заполняет текстовый параметр экземпляра "Номер помещения" для элементов (FamilyInstance) категорий "Мебель" и "Оборудование", определяя их пространственное положение относительно помещений (OST_Rooms), размещенных в связанной модели "АР". 

## Установка
1. Закрыть Revit.
2. Запустить `install.bat` из папки `build`.
3. Открыть Revit — вкладка с плагином появится на ленте.

## Требования
Revit 2020 - Revit 2024 (.NET Framework 4.8).
При просмотре .sln подключить ссылки на RevitAPI.dll и RevitAPIUI.dll из папки установки Revit.
